Overview

MachineState objects are slightly different from the other StepNC API objects- you need a unique MachineState object for every file you wish to simulate. Call the constructor with the path to the file you wish to simulate to instantiate an instance of it- ms = new MachineState("File.stpnc");

AdvanceState()

int AdvanceState ();

The AdvanceState() function advances the simulation by the smallest possible amount. This will move a tool along a toolpath. Returns something if it's at the end of a workingstep, I don't remember if it will switch workingsteps or not- will update.

NextWS()

void NextWS ();

The NextWS() function advances the simulation to the next sequential workingstep.
